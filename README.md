# Análisis de Desempeño de Tiendas

## Descripción del proyecto

Este proyecto tiene como objetivo analizar el desempeño de cuatro tiendas utilizando técnicas de análisis de datos. A partir de diferentes indicadores de negocio se busca identificar cuál tienda sería la mejor candidata para vender.

El análisis fue desarrollado en Google Colab utilizando Python para el procesamiento y visualización de los datos.

## Objetivo

Evaluar el rendimiento de cada tienda considerando:

- Ingresos totales
- Categorías de productos más vendidas
- Calificación promedio de los clientes
- Productos más y menos vendidos
- Costo promedio de envío

El objetivo final es generar una recomendación basada en datos para determinar qué tienda debería vender el Sr. Juan.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

## Análisis realizados

Durante el proyecto se realizaron los siguientes análisis:

- Cálculo de ingresos totales por tienda
- Identificación de categorías de productos más vendidas
- Análisis de satisfacción del cliente mediante calificaciones promedio
- Identificación de productos más y menos vendidos
- Cálculo del costo promedio de envío por tienda
- Visualización de resultados mediante gráficos

## Cómo ejecutar el proyecto

1. Abrir el notebook en Google Colab.
2. Ejecutar las celdas de código en orden.
3. Revisar los resultados y las visualizaciones generadas.

## Conclusión

A partir del análisis realizado se identificaron diferencias en el desempeño de las tiendas. La decisión sobre cuál tienda vender se basa en indicadores clave como ingresos, satisfacción del cliente y volumen de ventas.

El uso de análisis de datos permite tomar decisiones estratégicas fundamentadas y mejorar la gestión del negocio.
